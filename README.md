# AI Video Annotator

A powerful FastAPI-based backend and UI for AI-powered video annotation. This system processes uploaded videos to detect and track objects, recognize faces, extract audio transcripts, and generate searchable JSON metadata along with a browser-playable annotated video.

## Features
* **Object Detection & Tracking:** Powered by YOLOv8. Detects and tracks people, vehicles, animals, and other objects across video frames.
* **Face Recognition:** Uses DeepFace (FaceNet) to identify specific people in the video and log their attendance.
* **Audio Transcription:** Automatically extracts audio and transcribes spoken words using OpenAI's Whisper.
* **Web Dashboard:** Clean, responsive UI to upload videos, view playback, download JSON results, and search specific detected objects by label.

## Prerequisites
* **Python 3.10+**
* **FFmpeg:** Must be installed on your system for Whisper's audio extraction and OpenCV's video rendering.

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/John-cntr/ai-annotator.git
   cd ai-annotator
   ```

2. **Create and activate a virtual environment (Windows):**
   ```powershell
   python -m venv venv
   .\venv\Scripts\Activate.ps1
   ```

3. **Install dependencies:**
   ```powershell
   pip install -r requirements.txt
   ```
   *(Note: DeepFace and TF-Keras may need to be installed additionally depending on your environment).*

4. **Set up Face Recognition (Optional):**
   To use the automatic attendance system, create a folder named `known_faces/` in the root directory. Place clear `.jpg` or `.png` images of faces inside, naming the file as the person's name (e.g., `john_doe.jpg`).

## Usage

1. **Start the FastAPI server:**
   ```powershell
   uvicorn main:app --reload
   ```

2. **Access the Dashboard:**
   Open your web browser and navigate to: http://127.0.0.1:8000/

3. **Using the App:**
   * Click **Choose File** to upload an MP4, MOV, AVI, or WEBM video.
   * Click **Upload and Process**.
   * Wait for processing to complete. The terminal will display frame-by-frame progress. The first time you run this, it will automatically download the YOLO and DeepFace weights.
   * Once finished, the annotated video will play directly in the browser, and you can download the JSON metadata or view the Whisper transcript.

## API Documentation

Interactive API documentation (Swagger UI) is automatically generated by FastAPI.
When the server is running, you can access it at:
http://127.0.0.1:8000/docs
